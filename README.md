# Etch-A-Sketch
### 'Vanilla' JavaScript Learning Project

#### From: [The Odin Project](https://www.theodinproject.com/)

### This Project Makes Use of: 
- HTML
- CSS, CSS Grid, CSS Variables and Resets
- Dynamic Styling
- JavaScript EventListeners, Arrow Functions and DOM Manipulation
- Watching script values, CallStack, and inserting breakpoints were used to build and debug the code
